# Employee Misconduct Detection using YOLOv8

This Project detects employee misconduct such as:

- Drowsiness
- Mobile usage
- Crowding
- Mouth open
- Eye state monitoring

## Tech Used
- Python
- YOLOv8
- OpenCV
- Firebase
- Deep Learning models (.pt, .h5)

## How to Run
```bash
pip install -r requirements.txt
python app.py
